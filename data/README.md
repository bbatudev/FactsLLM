# Data Directory

This directory is reserved for large data files and indices required by the project.

Due to GitHub's file size limits, the actual data files are **NOT** included in the repository.

## Required Files
You should place the following files here after downloading them:

*   `fever_gold.index`
*   `teiyt_index.bin`
*   `wiki_index.bin`
*   `*.jsonl` datasets

See the main `README.md` for download links and setup instructions.
